📚 Sobre la Estructura BEM

Block (Bloque)

Un componente independiente y reutilizable. Ejemplo: .course-card es el bloque principal que contiene toda la tarjeta del curso.

Element (Elemento)

Partes del bloque que tienen sentido solo dentro de él. Ejemplos: .course-card__title, .course-card__price, .course-card__cta

Modifier (Modificador)

Variantes del bloque con comportamiento o estilo diferente. Ejemplos: .course-card--featured, .course-card--compact, .course-card--bestseller

Beneficios de usar BEM:

  • Modularidad: Cada componente es independiente y reutilizable
  • Escalabilidad: Fácil agregar nuevas variantes (modificadores) sin afectar el código existente
  • Mantenibilidad: Nombres claros hacen que sea fácil entender la relación entre clases
  • Colaboración: Los desarrolladores entienden la estructura sin confusiones
  • Prevención de conflictos: Las clases anidadas evitan colisiones de nombres CSS
  • Facilita el testing: Componentes bien aislados facilitan pruebas automatizadas
